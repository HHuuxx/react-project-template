# React-Project-Template

A template for creating a project using react

---

##### Modules Installed

| Module Name        | Version |
| ------------------ | :-----: |
| [react]            | 17.0.1  |
| [react-dom]        | 17.0.1  |
| [react-scripts]    |  4.0.0  |
| [react-redux]      |  7.2.2  |
| [node-sass]        | 4.14.1  |
| [react-router-dom] |  5.2.0  |
| [axios]            | 0.21.0  |
| [redux]            |  4.0.5  |
| [redux-thunk]      |  2.3.0  |
| [web-vitals]       |  0.2.4  |

##### Usage

1. Install packages after cloning the repository:

```
npm install
```

2. Start the development server:

```
npm start
```
